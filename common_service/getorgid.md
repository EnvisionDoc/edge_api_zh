# Get OrgId

获取当前组织ID。

## 请求格式

```
GET /connectService/orgId
```

## 响应参数

| **名称** | **数据类型** | **描述** |
|:---------|:-------------|:---------|
| orgId    | String       | 组织ID   |

## 示例 1

### 返回示例

```json
{
​        "status": 0,
​        "msg": "Success",
​        "submsg": null,
​        "data": {
​            "orgId": "XXXXXXXX"
​        }
}
```
